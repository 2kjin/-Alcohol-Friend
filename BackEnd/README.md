## 프로젝트 개요

Node.js + Typescript + NestJS 기반 백엔드 서버 프로젝트

## 세팅 하기

(package.json 위치에서 진행)

```bash
$ npm install
```

## 실행 방법

```bash
# development
$ npm run start

# watch
$ npm run start:dev

# production
$ npm run start:prod
```

## 유닛테스트 방법

```bash
# unit tests
$ npm run test
```
